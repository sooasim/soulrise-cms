# SoulRise Strapi CMS

SoulRise 웹사이트의 콘텐츠 관리 시스템입니다.

## 시작하기

### 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npm run develop
```

### 프로덕션

```bash
# 빌드
npm run build

# 프로덕션 서버 시작
npm start
```

## API 엔드포인트

- **Posts**: `/api/posts`
- **Pages**: `/api/pages`

## 관리자 패널

개발 모드: `http://localhost:1337/admin`
프로덕션: `https://soulrise-cms-production.up.railway.app/admin`
